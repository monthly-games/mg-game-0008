## 3. 상세 BM 설계

### 3.1 메타 성장 BM

#### 무료 구조
- 루프 종료 시 메타 자원 획득 (실패해도 지급)
- 웨이브 도달 수에 비례한 보상
- 일일 미션으로 추가 메타 자원

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 메타 자원 팩 | 대량 메타 자원 | $2.99~$9.99 |
| 메타 부스트 | 획득량 +50% (7일) | $1.99 |
| 전승 강화 | 루프 시작 보너스 강화 | $2.99 |
| 영구 버프 해금 | 특정 패시브 즉시 해금 | $1.99~$4.99 |

### 3.2 타워/유닛 BM

#### 무료 구조
- 기본 타워 5종 무료
- 업적/진행으로 추가 타워 3종 해금
- 이벤트에서 한정 타워 획득 기회

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 타워 해금 | 개별 타워 구매 | $1.99~$3.99 |
| 타워 번들 | 타워 3종 + 스킨 | $6.99 |
| 타워 스킨 | 외형 변경 (성능 동일) | $0.99~$2.99 |
| 유닛 팩 | 특수 방어 유닛 | $2.99~$4.99 |

### 3.3 루프 편의 BM

#### 무료 구조
- 일일 루프 횟수 무제한
- 기본 배치 프리셋 1개
- 실패 리포트 무료 제공

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 배치 프리셋 | 추가 저장 슬롯 +3 | $1.99 |
| 빠른 시작 | 이전 배치로 즉시 시작 | 무료 (기본) |
| 루프 속도 2배 | 빠른 진행 | 광고 or $0.99/영구 |
| 재도전 토큰 | 동일 조건 재도전 | 프리미엄 재화 30 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 메타 2배, 루프 속도 2배, 부활 1회 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $1.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 웨이브 무과금 클리어 가능
- 타워 간 성능 차이는 플레이 스타일 차이로 제한
- 메타 성장은 시간이 해결 (과금은 단축만)

### 과금 압박 최소화
- 루프 횟수 제한 없음
- 실패해도 메타 자원 지급으로 성장 보장
- 어려운 웨이브도 메타 성장으로 돌파 가능
