# MG-0008: Flappy Adventure UI/UX 스펙

> **게임명**: Flappy Adventure (플래피 어드벤처)
> **장르**: 하이퍼캐주얼 플라이어
> **카테고리**: Year 1 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 세로 비행 게임 표준
- 단순 탭 조작
- 최소 UI로 게임 영역 최대화

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 점프/비행 | 전체 화면 | 탭 |
| 일시정지 | 44×44dp | 탭 |
| 게임 시작 | 플레이 버튼 | 탭 |

### 터치 피드백
- 탭: 캐릭터 상승 애니메이션
- 코인 수집: 팝 이펙트 + 점수 증가
- 충돌: 화면 쉐이크 + 게임 오버
