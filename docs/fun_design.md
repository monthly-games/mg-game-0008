## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 마을 방어 디펜스 세팅/플레이                         │
│         ↓                                               │
│  2. 웨이브 진행 및 보상 획득                             │
│         ↓                                               │
│  3. 패배/시간 도달 시 루프 종료 → 메타 자원 획득          │
│         ↓                                               │
│  4. 루프 업그레이드 후 재도전 ─────────────→ (1로 반복)   │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **디펜스 세팅**: 타워/유닛 배치, 전략 설정
2. **웨이브 진행**: 적 웨이브 방어, 보상 획득
3. **루프 종료**: 패배 또는 시간 제한 도달 시 메타 자원 획득
4. **루프 업그레이드**: 영구 강화 후 더 나은 조건으로 재도전

## 3. 재미 축 (Fun Pillars)

### 3.1 재도전/학습 (Retry)
- **목표**: 실패해도 다음엔 더 잘할 수 있다는 희망
- **핵심 경험**: 이전 루프의 경험이 다음 루프를 쉽게 만드는 느낌
- **주요 요소**:
  - 사망 리포트
  - 패턴 학습
  - 메타 성장
  - 기록 경신

### 3.2 빌드/배치 (Build)
- **목표**: 최적의 방어 배치를 찾는 전략적 재미
- **핵심 경험**: 이 배치가 정답이었어! 하는 발견
- **주요 요소**:
  - 타워 종류/업그레이드
  - 배치 시너지
  - 지형 활용
  - 스킬 조합

### 3.3 점점 빨라지는 성장 (Acceleration)
- **목표**: 루프를 거듭할수록 빠르게 강해지는 체감
- **핵심 경험**: 첫 루프에선 10웨이브, 이제 50웨이브까지 가는 성장
- **주요 요소**:
  - 메타 업그레이드
  - 전승 요소
  - 해금 콘텐츠
  - 가속 아이템

## 4. 디자인 레버 (Design Levers)

### 4.1 전승 요소 설계
| 레버 | 설명 |
|------|------|
| 일부 구조/업그레이드 유지 | 루프마다 일부 구조/업그레이드가 남는 "전승 요소" 설계 |
| 구현 방향 | 핵심 타워 1개 유지, 패시브 버프 영구 적용, 시작 자원 증가 |

### 4.2 루프별 제약 조건
| 레버 | 설명 |
|------|------|
| 다른 목표/제약 | 각 루프에 다른 목표/제약 조건(타워 제한, 속성 제한 등) 부여 |
| 구현 방향 | 이번 루프: 화염 타워만 사용 가능, 다음 루프: 유닛 3개 제한 |

### 4.3 실패 보상 설계
| 레버 | 설명 |
|------|------|
| 의미 있는 실패 보상 | 실패 루프에도 의미 있는 보상 제공해 계속 돌고 싶게 설계 |
| 구현 방향 | 도달 웨이브 × 보상 배율, 실패 시에도 메타 포인트 획득 |

### 4.4 배치 공유 시스템
| 레버 | 설명 |
|------|------|
| 공략 발견 재미 | 다른 유저 배치 공유/추천으로 공략 발견 재미 제공 |
| 구현 방향 | 인기 배치 랭킹, 친구 배치 복사, 추천 빌드 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 루프 횟수 | 5회 이상 | 루프 시작 로그 |
| 평균 루프 시간 | 10~15분 | 루프 시작 → 종료 |
| 재도전률 | 90% 이상 | 실패 후 24시간 내 재시작 |
| 최고 웨이브 경신률 | 주 2회 이상 | 개인 기록 갱신 로그 |
| D7 리텐션 | 35% 이상 | 7일차 잔존율 |
