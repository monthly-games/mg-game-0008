## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 루프/전승 시스템 이해도 향상 (튜토/툴팁) |
| 7~10주 | 기본 맵/타워/적 세트 안정화 |
| 11~12주 | 첫 루프 챌린지 이벤트 준비 |

**핵심 과제:**
- 타임루프 시스템 튜토리얼 강화
- 초기 타워 밸런스 조정
- 전승 시스템 이해도 향상

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 루프별 특수 조건 챌린지 시작 |
| 5개월 | 새로운 맵 테마 추가 |
| 6개월 | 타워 제한 챌린지 |
| 7개월 | 새로운 적 테마 추가 |
| 8개월 | 속성 제한 챌린지 + 여름 이벤트 |

**핵심 과제:**
- 특수 조건 챌린지 사이클 정착
- 맵/적 테마 다양성 확보
- 루프 재도전 동기 부여

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 커뮤니티 공략/배치 공유 기능 |
| 10개월 | 0020(타임 슬립 탐험대) 설정/메타 연계 |
| 11개월 | 장기 루프 시즌 시작 |
| 12개월 | 1주년 이벤트 + 연말 레이드 |

**핵심 과제:**
- 커뮤니티 기능 강화
- 크로스 게임 세계관 연계
- 장기 시즌 시스템
