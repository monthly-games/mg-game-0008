## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **정밀 조작** | 정확한 타이밍의 탭 조작 | ★★★★★ |
| **기록 갱신** | 최고 점수 달성의 성취감 | ★★★★★ |
| **즉각적 재시작** | 실패 즉시 재도전 가능 | ★★★★☆ |
| **단순 명쾌함** | 규칙이 단순하고 명확함 | ★★★★☆ |

### 1.2 만족감 요소 (Satisfaction)
- **파이프 통과**: 장애물 통과할 때마다 점수 획득
- **연속 통과**: 다수 파이프 연속 통과의 긴장감
- **신기록**: 개인 최고 기록 갱신
- **메달 획득**: 점수별 메달 (브론즈, 실버, 골드, 플래티넘)

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 중력과 탭 조작의 미세 조정
├── 파이프 간격 변화
├── 하드 모드 (좁은 간격)
├── 점수 기반 메달 시스템
└── 친구 기록 도전
```

### 1.4 사회적 요소 (Social)
- **리더보드**: 글로벌/친구 점수 랭킹
- **기록 공유**: SNS 점수 공유
- **일일 챌린지**: 동일 조건 경쟁
- **메달 비교**: 친구와 메달 수 비교

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (15초~2분)
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [탭 시작] → [날기] → [파이프 통과]              │
│       ↑                      ↓                   │
│       └──── [게임 오버] ← [충돌] ←┘             │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **탭 시작** (즉시): 화면 탭으로 게임 시작
2. **날기** (지속): 탭으로 상승, 중력으로 하강
3. **파이프 통과** (지속): 파이프 사이로 통과하여 점수 획득
4. **충돌** (즉시): 파이프/땅 충돌 시 게임 오버
5. **게임 오버** (3초): 점수 확인, 재시작

### 2.2 세션 루프 (3~10분)
```
[접속] → [일일 보상] → [게임 시작] → [게임 오버] → [재도전/메뉴]
                             ↓                          ↓
                        [점수 확인]               [스킨 변경]
```

**세션 구성:**
- 일일 보상: 10초
- 게임 5~10회: 2~8분
- 스킨/업그레이드 확인: 1~2분

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [점수 달성] → [골드 획득] → [스킨 해금]              │
│         ↓                              ↓             │
│  [메달 수집]                    [업그레이드 강화]      │
│         ↓                              ↓             │
│  [더 높은 점수 도전] ←──────────────────┘            │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[메인 메뉴] ────→ [스킨 선택]
    │             [상점]
    │             [리더보드]
    │             [설정]
    ↓
[PLAY] → [게임 화면] → [탭하여 시작]
                          ↓
                     [게임 플레이]
                          ↓
                     [게임 오버]
                          ↓
                     [점수/재시작]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **리워드 광고** | 50% | 보상 2배, 부활 기회 |
| **스킨** | 25% | 새 스킨, 파이프 스킨 |
| **광고 제거** | 15% | 일회성 구매 |
| **인터스티셜** | 10% | 게임 오버 후 광고 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 전체 게임 플레이 가능
├── 기본 스킨 사용
├── 광고 시청으로 보상 2배
└── 일일 무료 골드 보너스

과금 유저 혜택:
├── 광고 제거 ($1.99)
├── 스킨 팩 ($0.99~$2.99)
├── 골드 패키지 ($0.99~$4.99)
├── 스타터 팩 ($2.99)
└── 전체 스킨 해금 ($6.99)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 소량 골드 (500) | $0.99 | 스킨 1개 |
| 중량 골드 (2500) | $2.99 | 20% 보너스 |
| 대량 골드 (6000) | $4.99 | 40% 보너스 |
| 광고 제거 | $1.99 | 영구 적용 |
| 스킨 팩 | $0.99~2.99 | 개별 스킨 |
| 올 스킨 팩 | $6.99 | 모든 스킨 해금 |

### 3.4 리텐션 메커니즘
- **일일 보상**: 출석 체크 (7일 사이클)
- **일일 퀘스트**: 점수 달성, 게임 횟수
- **주간 챌린지**: 특수 조건 달성
- **시즌 스킨**: 한정 기간 스킨

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
파이프 간격
  ▲
넓음 │────╮
     │    ╰────────────────────
좁음 │
     └──────────────────────────→ 점수
        0   10   20   30   40
```

**노멀 모드 - 고정 간격**
**하드 모드 - 점수에 따라 간격 감소**

**난이도 공식 (하드 모드):**
```
파이프_간격 = max(최소_간격, 기본_간격 - 점수 × 2)
파이프_속도 = 기본_속도 × (1 + 점수/100 × 0.1)
```

### 4.2 콘텐츠 진행
| 점수 | 메달 | 해금 |
|------|------|------|
| 0-9 | - | 기본 플레이 |
| 10-29 | 브론즈 | 스킨 1개 해금 |
| 30-49 | 실버 | 하드 모드 해금 |
| 50-99 | 골드 | 특별 스킨 해금 |
| 100+ | 플래티넘 | 전설 스킨 해금 |

### 4.3 밸런싱 공식
```dart
// 탭 물리 계산
void onTap() {
  bird.velocity = -flapPower; // 위로 점프
}

void update(double dt) {
  bird.velocity += gravity * dt; // 중력 적용
  bird.position.y += bird.velocity * dt;
}

// 점수별 보상
int goldReward(int score) {
  return score * 5 + getMedalBonus(score);
}

int getMedalBonus(int score) {
  if (score >= 100) return 500;
  if (score >= 50) return 200;
  if (score >= 30) return 100;
  if (score >= 10) return 50;
  return 0;
}

// 업그레이드 효과
double getFlapBoost(int upgradeLevel) {
  return 1 + upgradeLevel * 0.05; // 레벨당 5% 증가
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 점수 골드 | 게임 종료 | 점수×5 |
| 메달 보너스 | 메달 획득 | 50~500 |
| 일일 보너스 | 출석 | 50~200 골드 |
| 퀘스트 보상 | 완료 | 100~500 골드 |
| 업적 보석 | 마일스톤 | 10~100 보석 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 현재 점수 (상단 중앙)
├── 일시정지 버튼
└── 준비 상태 표시

레벨 2 (상호작용 시):
├── 게임 오버 점수판
├── 메달 표시
└── 재시작/메뉴 버튼

레벨 3 (팝업):
├── 신기록 연출
├── 설정 메뉴
└── 상점
```

### 5.2 네비게이션 흐름
```
[메인 메뉴]
    ├── [PLAY] → [게임] → [게임 오버] → [재시작/메뉴]
    ├── [스킨] → [스킨 선택/구매]
    ├── [상점] → [골드/보석]
    ├── [리더보드] → [랭킹]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 탭/날기 | 날개 애니메이션 | 날개짓 소리 | - |
| 점수 획득 | 점수 증가 | 점수음 | - |
| 메달 획득 | 메달 팝업 | 메달음 | 진동 (약) |
| 충돌 | 충돌 이펙트 | 충돌음 | 진동 (강) |
| 신기록 | 신기록 배너 | 팡파레 | 진동 (중) |

### 5.4 접근성
- **원탭 조작**: 탭만으로 모든 조작
- **큰 터치 영역**: 화면 전체 탭 인식
- **색맹 모드**: 파이프 패턴 추가
- **효과 감소**: 시각 효과 줄이기
- **세로 모드**: 한 손 플레이 최적화
